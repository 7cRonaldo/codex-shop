# 山下饭堂（Vue 3）

使用 Vue 3、TypeScript 和 Vite 构建的本地静态餐厅网站。

## 本地运行

需要 Node.js 20.19 或更高版本。

```bash
npm install
npm run dev
```

浏览器访问终端显示的本地地址即可。项目不依赖 Cloudflare、Wrangler、OpenAI Sites、私有认证或其他在线托管服务。

## 命令

- `npm run dev`：启动本地开发服务器
- `npm run build`：类型检查并生成生产文件到 `dist/`
- `npm run preview`：本地预览生产构建
- `npm test`：运行测试
